The Suite Components

Two specialized, robust plugins working in harmony to secure and guide agentic executions.

agent-pipeline-antigravity

Structured Multi-Agent Runs & Controls

A rigorous orchestration engine that executes complex developer tasks across dedicated stages: Research, Plan, Execute, Verify, and Critique. It coordinates the lifecycle seamlessly without losing context.

  • 11 Lifecycle Hooks: Intercept and validate agent operations (e.g. PreToolUse, SessionStart) to prevent scope drift.
  • Directive-Contract Cryptography: Signs task requirements with SHA-256 hashes to automatically verify sub-steps.
  • Compaction-Resistant Memory: Saves state to local cache before LLM compactions, and restores it instantly post-compaction.
  • Mem0 Docker Continuity: Integrate with Mem0 cross-session memory platforms to remember details across restarts.

Zero Context Loss

LLM context compactions wipe out short-term variables. Our hook engine writes state cache to disk and restores it dynamically upon session reload, allowing workflows to run indefinitely without forgetting goals.

Drift & Edit Boundaries

Ensure the agent never edits files outside the approved directory or runs dangerous terminal commands. The PreToolUse hook screens and restricts all model tool activities dynamically.

audit-skills-antigravity

Consensus-Based Review & Verification

Elevate code quality using automated, role-based audits. Rather than relying on simple heuristics, these tools generate evidence-based audits complete with file locations, line references, and severe rollup logs.

  • Audit Lite: Quick, single-pass spot checks for small patches, bug fixes, or minor features, validating tests, docs, and code.
  • Audit Team: Simulates a 5-role expert panel (Security Expert, Lead UX, Staff Engineer, Technical Writer, Test Lead) to examine your changes.
  • Evidence-Based Validation: Every defect requires precise citations (files, line ranges, outputs) ensuring zero hallucinations.
  • Definition of Done (DoD) Checks: Standardized rollups into Blocker, Critical, Major, Minor, and Nit severities.

5-Lens Simulation

Simulate specialized tech experts debating your PR. The Security Expert flags unsafe API usage while the Lead UX inspects accessibility, providing a holistic feedback loop before merging.

Actionable Reports

Generates rich, clean markdown audit reports directly in your workspace folder. Use reports as verification evidence for CI/CD gates or PR approvals.

Suite Architecture

How the plugins interact with the Antigravity Shell and your local workspace files.

Execution Hook Sequence

1
SessionStart

Loads active run variables and starts Mem0 docker container.

2
PreToolUse & PermissionRequest

Intercepts file edits and runs scope checks against scope-lock.yaml.

3
PreCompact / PostCompact

Locks state variables to disk and feeds them back into context to survive token resets.

4
SessionEnd

Updates Mem0 database memory tables and clears run-locks safely.

Setup & Deployment

Quickly copy the plugins into your global configuration folder and verify them.

Quick Start

1

Clone the Monorepo

Download the latest release and plugin directory structure.

2

Copy plugins to configuration

Move the files into your system's global Antigravity installation folder.

3

Validate Installation

Run standard unit tests with pytest to confirm environment health.

PowerShell 
# Clone the repo
git clone https://github.com/scottconverse/agent-pipeline-antigravity-suite.git
cd agent-pipeline-antigravity-suite

# Copy plugins to global user directory
Copy-Item -Path "plugins/*" -Destination "$HOME/.gemini/config/plugins" -Recurse -Force

# Run test suite verification
cd plugins/agent-pipeline-antigravity
python -m pytest -m "not cleanroom_e2e"
Bash
